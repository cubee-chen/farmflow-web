## 3. RLS 隔離測試

**測試步驟**：

1. 用陳惠茹（`chenhuiru@farmflow.local`）登入
2. 建立或記下一筆訂單 UUID（稱為 ORDER_ID）
3. 登出
4. 用徐方（`xufang@farmflow.local`）登入
5. 訪問 `/orders/{ORDER_ID}`

**預期結果**：
- `/orders/{ORDER_ID}` 顯示「找不到此訂單」或 404（RLS 讓徐方看不到陳惠茹的訂單）

**技術機制**：
- RLS policy `farmer_owns_orders`：`USING (farmer_id = current_farmer_id())`
- `current_farmer_id()` 從 `auth.uid()` 查 `farmers.auth_user_id` 取得
- 徐方的 `auth.uid()` 對應 `current_farmer_id()` = 徐方的 farmer.id
- 查詢陳惠茹的訂單（farmer_id = 陳惠茹的 id）→ RLS 過濾 → 0 rows → 404

**狀態**：待手動執行（需兩組帳號切換測試）
