# P0.5 Known Issues (Non-blocking)

## KI-1: Signed URLs expire after 1 hour

**Scope:** `ImageGallery` in order detail and `ImagePreviewSection` in draft editor.  
**Behaviour:** If the page is left open and the signed URL TTL (3600s) passes, thumbnail/modal images will return 403. Refreshing the page or re-opening the collapsible reloads the URLs.  
**Workaround:** User re-opens the card. No data is lost.
